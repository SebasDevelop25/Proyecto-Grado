    TYPE()


    Esta función sirve para saber de que esta compuesto un elemento, es decir se utiliza para saber, si lo que esta dentro del elemento, es un texto (string), un número, ya sea decimal o no, un valor booleano (verdadero o falso), una lista, etc...


    Sintaxis

                            
                    print(type(Texto o Número))


    Recuerda que en esta funcíon solo puedes poner textos, números, valores booleanos, etc, pero por separado es decir, no los puedes juntar 


                            
   print(type("Hola mundo"))
      
   print(type(24))

   print(type(24.6))

   print(type(True))

   print(type([2 , 4]))

   print(type((6 , 14)))
                        
                    


                        
             class 'str' = string(cadena de texto)

             class 'int' = entero(número)

             class 'float' = decimal(número)

             class 'bool' = booleano(valor lógico, true o false)

             class 'list' = lista(valores agrupados)

             class 'tuple' = lista(valores únicos)

    TUPLES


    Es similar a las listas, sus únicas diferencias es que la tuples utilizan valores únicos, es decir estos no se pueden modificar y se abren y se cierran con ().


    Ejemplo
